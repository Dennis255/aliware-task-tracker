# Aliware Task Tracker

Prueba técnica Full-Stack implementada con Quarkus (Backend) y Vue 3 (Frontend).

## 🚀 Tecnologías
- **Backend:** Java 21, Quarkus, Hibernate Panache, JWT, Flyway.
- **Frontend:** Vue 3, Vuetify, Pinia, Axios.
- **Base de Datos:** PostgreSQL.
- **Infraestructura:** Docker & Docker Compose.

## 📋 Requisitos Previos
- Docker y Docker Compose instalados.
- (Opcional) Java 21 y Node.js 22 para ejecución local.

## 🛠️ Ejecución Rápida (Docker)
Para levantar todo el entorno (Base de datos + Backend + Frontend):

1. Clona el repositorio.
2. Ejecuta en la raíz del proyecto:
   ```bash
   docker-compose up --build
